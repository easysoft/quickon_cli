## qcadmin init

Run this command in order to set up the QuCheng control plane


### Examples

```
 			--svcsubnet "10.43.0.0/16" \
			--eip 1.1.1.1  \
			--san kubeapi.k8s.io

	create ha cluster with mysql 5.7
		q init --podsubnet "10.42.0.0/16" \
			--svcsubnet "10.43.0.0/16" \
			--eip 1.1.1.1  \
```
